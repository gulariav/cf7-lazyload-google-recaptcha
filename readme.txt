== Description ==

This plugins use for your website speed improvement and decrease your page request. When you have used contact form 7 and insert you Google Captcha( v3 ) after this plugin active. When a user scrolls the page, then loading google captcha code. This plugin based on "cf7-lazyload-google-recaptcha" by amit648.

How to use this plugin?

1. Download this Plugin.
2. Upload it through plugin uploader and activate.
3. Once you install this plugin successfully, click on  Activate Plugin .
4. Add this css class "fire-recaptcha-onfocus" to one any required field.

Note: This plugin required integration with Google Recaptcha, so first add captcha keys.




== Installation ==

**Installation**

1. Upload the entire 'CF7 Lazyload Google Recaptcha' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
